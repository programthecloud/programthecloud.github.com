Overview

CountVoteCounter

CountVoteCounter is an implementation of the VoteCounterProtocol in which a certain number of required votes for a “winning” candidate is provided directly.

- (Object) winning_vote([:ballot_id], :vote)

Scratch to hold the winning vote of a completed ballot, if one exists. Note: There can only be one winner for a ballot. Duplicate key error will be thrown if num_required is set improperly such that there can be multiple winners. This constraint stems from the fact that the output interface result has [:ballot_id] as its key, indicating at most one winner per ballot_id.
